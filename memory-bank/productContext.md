# Product Context

## Why this project exists
People take notes the way their thoughts arrive — messy, unstructured, multiple topics tangled together. Reading those notes back later is often harder than not having taken them at all, and organizing them properly takes time most people don't make. This product removes that step: the user writes freely, and AI does the organizing afterward.

## Problems it solves
* Raw, unstructured notes that are unreadable after the fact
* The "I'll organize this later" task that never actually happens
* Mixed-context note-taking (e.g. a grocery list with errands and reminders tangled together) that's hard to act on as written

## How it should work
The user lands on the app and starts writing immediately — no setup, no account required to try it. Anonymous use is purely a scratchpad: write, save, view, gone when the tab closes. To get the actual value of the product — AI-organized, categorized notes that persist — the user creates an account (email + password, verified by email). From that point, every note they save is automatically classified into one of a small set of templates (e.g. shopping list, meeting notes, lecture notes, daily plan, travel list) and restructured into a clean, readable format, then stored permanently and viewable from a history sidebar.

## User Experience Goals
* **Frictionless first impression:** no signup wall before trying the core writing experience
* **Calm & uncluttered:** the UI should never feel busy — the product's purpose is to reduce mental clutter, so the interface has to model that
* **Clear value moment:** the gap between "plain note" (free) and "AI-organized note" (account required) should be obvious and convincing without needing AI to be visible before signup
* **Fast feedback:** AI processing should feel close to instant; loading states should never feel like a stall
* **Minimal cognitive load:** clear hierarchy, consistent spacing and typography, no decorative complexity
